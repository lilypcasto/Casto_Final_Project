# Casto_Final_Project

This is the repo for the Final Project for Data Wrangling.

## **Dataset Summary:** 

#### The repository includes code for the documentation of Canadian active wildfires in 2025.

**File Structure:**\
- `data_raw/` – raw datasets
- `code/` – R code and documents
- `outputs/` – cleaned and finalized data, figures, tables, and reports
- `README.md` – overview of the project

**Data Set Contents:** The data set contains information on the
following:

1.  `agency type responding to the fire`
2.   `fire-name classification`
3.  `latitude and longitude of the fire`
4.  `hectares burned`
5.  `exact start date`
6.  `stage of control`
7.  `response type`
